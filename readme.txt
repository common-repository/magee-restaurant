== Description ==

### Overview 

Magee Restaurant plugin is an easy-to-manage modern solution for building online menus of restaurants, cafes and other typical food establishments. It is built with woocommerce, which can help you sell food and drinks online without any technical help. The plugin uses default WordPress functionality. You can create and edit menu items inside the Magee Restaurant Options, use a shortcode to display it in posts and pages. It is also the perfect plugin for receiving, managing and handling bookings easily. 


### Plugin Features 

* Fully responsive and mobile ready.
* Use a shortcode to display menu in posts and pages.
* Variable options and index setting.
* Option setting like position and title etc.customers can choose For here or to go.
* Built with woocommerce contributes import of relevant product data .
* Primary color setting or more.
* Each shortcode represents a style
* Shopping cart
* Control the shopping cart display page with Settings
* Control thumbnail image with Settings

### Instructions

* You need to install woocommerce before you use Magee Restaurant
* Install plugin.
* Add new menu in magee restaurant.
* Control Settings.
* Use the shortcode to you page or post. 

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/magee-restaurant/*'

Activate plugin at "Plugins" administration page.
